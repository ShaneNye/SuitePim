Alpha Release Notes | 1.0.13:

This Platform serves as a Management Tool for Lot Numbered Inventory Items within the Sussex Beds Netsuite Enviroment for the Product Development & Website Teams.

To access the menu hover your mouse over the blue Header

Product Development Testing: Please go through all the processes you’d normally complete using the Live Spreadsheet. If you come across anything you can’t do, or notice any features that aren’t working as expected, just raise a support ticket and I look into it. It would be great if all testing could be wrapped up by 22/09, as a final development period will begin then, ready for final review and full release by 01/10.

Current Feature Status:
  • Product Data :
      • This feature is a replacement for the product development teams live spreadsheet exsisting items can be searched, reviewed, updated and pushed back to netsuite. black columns are 'Dynamic' - they do not import into netsuite but they give the user an indication of supporting information such as 'Retail Price' & 'Margin'
  • Product Validation :
      • This feature allows users to quickly see inventory items that are missing standard field configuration that may cause issues with items being processed correctly.
  • Request Support :
      • This Feature allows users to raise & manage SuitePim support requests which is linked to jira & Git for support team management.

Once Fully complete & Deployed a full user guide will be available for the Sussex Beds | SuitePim
